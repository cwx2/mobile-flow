Quick Start

1

Install Agent on your computer

Download from Releases → run it → a tray icon appears with your IP and password.

2

Install App on your phone

Download APK from Releases → install → enter IP, port 9600, and password.

3

Install an AI CLI tool

npm i -g @anthropic-ai/claude-code (or Codex, Gemini CLI, Kiro, etc.)

4

Start coding!

Open the app, connect to your desktop, and code from anywhere.
